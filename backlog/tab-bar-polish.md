# Tab Bar Polish

Further refinement of main navigation tabs (Library / Queue).

## Current direction

Flat rectangular tabs — no pill radius, no button-like padding, centered label text, active tab uses content background.

## Ideas

- Icons alongside labels (optional)
- Underline or accent indicator variants
- Keyboard focus and aria improvements
- Drag-to-reorder tabs if more views are added
- Badge styling for queue count
- Compact mode on narrow windows

## Notes

Initial simplified tab styling landed outside Phase 5. Use this story for iterations after trying the app.
