# Native Polish

OS-level desktop integration. UI look-and-feel is covered in **Phase 5** of [`PLAN.md`](../PLAN.md).

## Ideas

- Keyboard shortcuts (space = play/pause, arrows = seek, etc.)
- System integration: media keys, dock/taskbar progress (macOS)
- Settings: default import folder, volume, theme tweaks
- Mini player mode

## Touches

- `app` layer, `playback` feature, Tauri plugins
