# Library Organization

Browse and categorize media beyond a flat list.

## Ideas

- Categories / tags with filter support
- Sort by title, artist, date added, duration
- Add library items to queue (reference, not duplicate file)
- Remove from queue vs remove from library
- Browse views: by folder, artist, album (from metadata)
- Bulk actions: multi-select, batch tagging

## Touches

- `library` feature (model, filters, UI)

## Depends on

- Phase 2 (playback)
- Phase 3 (files & persistence)
- Phase 4 (queue) — overlaps with playlist-in-library model
- Phase 5 (UI polish) — optional; new views benefit from shared components
